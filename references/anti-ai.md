# 去 AI 味：从检查清单到底层认知

> AI味的本质不是某几个词或句式。它是一种**过度的秩序感**——人类的文字是混乱的、偏执的、不完美的，而AI的文字像一个穿着打扮完美但眼神空洞的人。

---

## 第零层：为什么AI写的东西"像AI"

AI文本的根本问题不在表面（用词、句式），而在**底层概率分布**：

1. **均值回归**：AI倾向于生成"最可能的下一个词"，导致所有输出都朝平均值靠拢。人类写作有极端值——特别好或特别烂的句子共存。
2. **对称性**：AI喜欢对称结构（"不是A而是B""既有A也有B"）。人类写作是不对称的——偏向某一边，忽略另一边。
3. **情感正确性**：AI的情感反应总是"合理"的。人类的情感反应经常是错位的、矛盾的、不成比例的。
4. **无损信息传递**：AI的每句话都携带信息，没有"废话"。人类写作充满真正的废话——而这些废话恰恰是"人味"的核心。
5. **视角稳定性**：AI的第一人称叙述像上帝视角——观察力完美、描述精准。人类的第一人称有盲区、偏见、自我欺骗。

**理解了这五个根因，你就能自己发明无数种去AI味的技法。**

## 第一层：句子级病症

### 必杀句式（出现即改，零容忍）

```
❌ "不是A，而是B"
❌ "与其说A，不如说B" 
❌ "这不仅仅是A，更是B"
❌ "某种意义上来说"
❌ "在这个A的时代"
❌ "值得一提的是"
❌ "不得不说"
❌ "毫无疑问"
❌ "A，而非B"
❌ "这一刻，他/她终于明白了……"
❌ "像是在说给自己听"
❌ "仿佛在无声地诉说着什么"
❌ "空气中弥漫着……的气息"
❌ "时间仿佛在这一刻静止了"
❌ "他/她的眼中闪过一丝……"
❌ "心中涌起一股难以言喻的……"
❌ "XX如同XX一般，XX着XX"（隐喻+动名词的标准AI句式）
❌ "他/她不禁……"
❌ "一种说不清道不明的感觉"
❌ "那一瞬间"（AI特别爱用"瞬间"）
❌ "某种程度上"
❌ "事实上"作为段落开头
❌ "然而"作为段落开头（全文最多出现2次）
```

### 为什么这些句式有AI味？

因为它们都是**高概率序列**——在AI的训练数据里出现频率极高，所以AI特别爱生成。人类也会用，但频率低得多。当这些句式在一篇文章里出现3次以上，就像一个人脸上有3颗一模一样的痣——你知道那是假的。

## 第二层：段落级病症

### 1. 段落节奏均匀（最致命的AI指纹）

**检测方法**：数每段的句子数。如果80%的段落都是3-5句话，这就是AI。

**人类的段落分布**应该像这样：
```
1句 | ████████████  (20-30%)
2句 | ████████      (15-20%)
3句 | ██████████    (20-25%)
4句 | ████          (10%)
5句 | ███           (5-8%)
6句+| ██            (偶尔出现)
```

**AI的段落分布**像这样：
```
1句 | ██            (5%)
2句 | ████          (10%)
3句 | ████████████  (35%)  ← 集中在中间
4句 | ████████████  (30%)  ← 集中在中间
5句 | ████          (10%)
6句+| ██            (5%)
```

**治法**：写完后强制打散。把长段拆成独句段。把连续的短段偶尔合并。制造**不均匀**。

### 2. 情感弧度太平滑

AI写的情感变化像正弦波——平滑、可预测。人类的情感变化像心电图——有突然的尖峰和不规则的平台。

```
AI情感:  ～～～∿∿∿～～～∿∿∿    （平滑波动）
人类情感: ——————╱╲___————╱╲╲╲╱—  （突然炸开，然后不规则恢复）
```

**治法**：
- 情绪爆发要**突然**，不要"慢慢积累到终于忍不住"
- 爆发后的恢复要**不规则**——不是马上冷静，而是余震式的起伏
- 允许**错位情绪**：该哭的时候笑、该愤怒的时候发呆

### 3. 因果链太完整

AI的每个段落都有清晰的逻辑连接：A因为B，B导致C，C所以D。

真人写作的因果链是**有断裂的**：A发生了。然后不知道为什么想到了X。对了说回C。

**治法**：
- 允许叙述者跑题——讲到一半突然想到另一件事，说完再绕回来
- 允许没有因果的并置——两件事放在一起，不解释为什么
- 允许"死线索"——提到的某个细节最后没用上

## 第三层：全文级病症

### 1. 信息分布太均匀

AI在全文中均匀地分配信息——每1000字揭露差不多同样多的新信息。

人类写作的信息分布是**不均匀的**：有的段落信息密度极高（一段话翻转整个故事），有的段落几乎没有新信息（角色在发呆、在做无关的事）。

**叶小白的信息分布**：
```
章节 1-3:  ██        （基本设定，密度低）
章节 4-7:  ████      （第一层谎言，中等密度）
章节 8:    █         （过渡）
章节 9:    ████████  （枪花被质疑，密度骤增）
章节 14:   ██████████（真相揭示，密度爆炸）
章节 19-20:█         （松弛尾声，密度极低——然后一句话引爆）
```

**治法**：信息密度要像鼓点，不像白噪音。有密有疏，疏的地方恰恰是蓄力。

### 2. 角色语言同质化

AI写5个角色说话，去掉名字分不清谁是谁。因为AI的所有输出都来自同一个概率模型。

**治法——角色DNA表**（每个角色写作前必须填）：

| 维度 | 角色A | 角色B |
|------|-------|-------|
| 句子平均长度 | 5-8字（断句型） | 15-25字（长句型） |
| 用词档次 | 粗糙（操、妈的、得了吧） | 文雅（确实、也不是不行） |
| 口头禅 | "我跟你说" | "你猜怎么着" |
| 说话习惯 | 爱打断别人 | 总是先沉默3秒再回答 |
| 信息泄露方式 | 直说 | 绕弯子暗示 |
| 情绪表达 | 外放（摔东西、骂人） | 内收（沉默、转身走） |

**检测方法**：遮住所有对话标签，随机抽5段对话。如果分不清谁说的，角色就没写好。

### 3. 叙述者"全知"偏差

AI写第一人称时，叙述者对环境的观察能力惊人——注意到每一个细节，每一个表情变化，每一个隐含意义。

真人作为叙述者是**有限的**：
- 会**漏掉**明显的事（因为当时在走神/太紧张/不关心）
- 会**误解**别人的意图（因为偏见/信息不足）
- 会**过度关注**无关的事（因为个人执念/焦虑转移）
- 会**事后重构**记忆（"现在想起来，当时他的眼神好像就有点不对"——这是事后归因，不是真的观察到了）

**治法**：
- 叙述者不应该注意到所有伏笔——有些伏笔是读者注意到但叙述者没注意到的
- 叙述者的判断可以是错的——"我当时觉得他是在心疼我。后来才知道不是。"
- 叙述者可以承认自己"记不清了"——"那天她穿什么来着？好像是红色的？反正不重要。"

## 第四层：气质级——最难模仿的"人味"

### 1. 不完美的美感

人类的好文字有一种**粗粝感**——像手工陶器上的指纹，像老照片的颗粒感。AI的文字是光滑的、完美的、无机的。

怎么制造粗粝感？
- **偶尔用错词**（不是真的错，而是用了一个"差不多"但不精确的词——因为真人写作时不会每个词都推敲）
- **句子偶尔写拧了**——语序不标准，但意思到了（"那天的雨大得，路上一个人没有"）
- **偶尔重复**——真人写作会不自觉地重复某个词（"那天很安静。安静得不对劲。"）
- **偶尔口语化到"不规范"**——"就……怎么说呢""反正就那么个意思"

### 2. 偏执和执念

AI是公正的、平衡的。人类是偏执的。

好的叙述者对某些事有不成比例的关注：
- 反复提到同一个细节（"她手上那个戒指"反复出现5次）
- 对某些角色有明显的偏见（描述讨厌的人时用词尖刻，描述喜欢的人时用词温柔）
- 对某些事有不合理的执念（"我就是想知道那天晚上他去了哪里"——即使这对故事推进不重要）

### 3. 幽默和自嘲

AI很少真正搞笑。因为幽默的本质是**期望违反**——你以为我要说A，我说了B。AI倾向于满足期望而不是违反它。

盐选爆款中的幽默通常是**黑色幽默**或**自嘲**：
- "我妈要是知道我抽烟肯定骂我。可惜她已经死了。"（黑色幽默）
- "我他妈居然信了。"（自嘲）
- "如果我再聪明一点，大概就不会站在这个派出所里了。"（自嘲+黑色幽默）

### 4. 沉默和留白

AI害怕沉默——它总想填满每一个空间。人类的好文字敢于留白。

```
# AI版
他听完后沉默了很久，似乎在消化这个消息的冲击。终于，他开口了，声音里带着一丝不易察觉的颤抖。

# 人类版
他没说话。
过了很久。
"走吧。"
```

沉默本身就是表达。不需要解释"为什么沉默""沉默中在想什么"。**解释沉默就是杀死沉默。**

## 实战对照表：同一段落的三个版本

**场景**：主角发现母亲藏在枕头里的存折

### AI版（0分）
```
我的手微微颤抖着，翻开了这本存折。映入眼帘的是一行行整齐的数字，从2009年开始，每个月都有一笔存入。有时候是五百，有时候是八百。看着这些数字，我的眼眶不禁湿润了。这不仅仅是一本存折，更是母亲十五年来无声的爱。每一笔存款，都承载着她对我深沉而厚重的牵挂。
```
**问题**：形容词堆砌、"不禁"、"不仅仅是A更是B"、结尾升华、情感直说

### 改良版（60分）
```
存折是绿色的，邮政储蓄银行，封面卷了边。

翻开。

户名：刘桂芬。开户日期：2009年8月17号。

第一笔存入：500元。

然后每个月都有一笔。500，800，有时候1000。最后一笔是今年七月。800元。

余额：247,500元。

我坐在那张硬板床上，愣了很久。
```
**进步**：段落短了、用具体数字代替形容、情感不直说。**但还是太"干净"了。**

### 爆款版（90分）
```
存折是绿色的。邮政储蓄那种，封面卷了边，看着有年头了。

翻开。

户名：刘桂芬。
开户：2009年8月17号。
她离婚后第三天。

第一笔：500。

然后每个月都有。500，800，偶尔1000。有一两个月断了，很快又接上。

最后一笔。今年七月。800。住院前一个月。

翻到最后一页。

二十四万七千五。

我坐在我妈那张一米二的硬板床上，拿着那个存折，愣了很久。

我妈一个月退休工资三千二。交完房租剩七百。

七百。

而她每个月还要从里面省出五百到一千。

也就是说她每个月花在自己身上的——吃饭、坐公交、买药——可能不到两百块。

我的眼泪一下子就下来了。

不是因为钱。二十四万七千五——以北京的标准，不是大数字。我一年工资就比这多。

是因为她从来没跟我说过。

十五年。

一个人，每个月从牙缝里挤出几百块，攒了十五年。
```
**为什么好**：
- "一米二的硬板床"——一个细节定场景
- "她离婚后第三天"——一句话给出情感重量，不解释
- "七百。"——独字段落，留白
- "不是因为钱"——叙述者主动纠偏，像真人在说话
- "从牙缝里挤出"——口语化表达
- 没有一句话在"升华"

## 终极检测流程

写完全文后依次执行：

### A. 句式扫描（2分钟）
- Ctrl+F 搜索本文"必杀句式"列表中的每一个 → 出现即改
- 数"的"字 → 连续出现2个以上的句子全部重写
- 数"然而/但是/不过/事实上" → 全文不超过3个

### B. 段落结构扫描（3分钟）
- 数每段句子数 → 独句段占比低于25%就加独句段
- 检查连续3段以上长度相似的 → 打散
- 检查有没有超过5句话的段落 → 拆开

### C. 对话扫描（3分钟）
- 遮住人名 → 能否分辨谁在说话？
- 有没有角色说了超过40字的完整长句？→ 拆开或改成断句
- 有没有"他语重心长地说""她坚定地回答"这类标签？→ 删掉

### D. 情感扫描（2分钟）
- 找到所有情感高潮点 → 有没有用"不禁""终于明白""泪水夺眶而出"？→ 改成动作或沉默
- 结尾是不是在"升华人生道理"？→ 删掉最后一段试试
- 情感反应是不是都"合理"？→ 加一个错位反应

### E. 贴吧测试（最终）
随机抽全文的3段，想象匿名发在贴吧/豆瓣。
- 会不会有人说"AI写的"？
- 如果有1%的可能 → 那段必须改
