# 草拟知乎回答

## 输入
$ARGUMENTS - 知乎问题标题或链接

## 任务
1. 如果是链接，用 Chrome MCP 打开并获取问题详情
2. 读取 `claude.md` 了解作者的专业领域和写作风格
3. 搜索博客目录 `/Users/geyunfei/dev/yfge/yfge.github.com/_posts/` 找到相关文章
4. 根据博客内容和作者风格，草拟一份知乎回答

## 回答风格
- 开门见山，先给结论
- 用实际案例说话（引用博客中的项目经验）
- 技术内容要落地可执行
- 适当加入个人观点和思考
- 结尾可以引导关注公众号 "IT老拐瘦" 或博客 yfge.github.io

## 输出格式
```markdown
## 回答草稿

[回答正文]

---
## 引用的博客文章
- 文章1: 链接
- 文章2: 链接

## 建议的标签
- 标签1
- 标签2
```

## 注意事项
- 保持"老拐"的风格：务实、不油腻、有干货
- 如果问题与专业领域不匹配，诚实告知并建议跳过
